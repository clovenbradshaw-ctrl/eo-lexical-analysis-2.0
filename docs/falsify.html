EO 3×3×3 Falsification Panel

Tests whether ANY 3×3×3 partition of the same embeddings reproduces the monotonic distance-by-axis-difference pattern attributed to EO. If random or surface-confound labels also produce monotonicity, the EO signature is an artefact of binning. If only EO and label-from-geometry methods produce it, EO is tracking real semantic structure.
Equivalent to falsify_3x3x3.py but runs entirely in your browser. Defaults to a 1500-clause subsample for speed; raise the cap if you have RAM and patience.

Inputs

Three ways to load data, in priority order: (1) URL params — append ?embeddings=URL&classified=URL&raw=URL to this page; (2) Auto-load from same directory — drop this file next to embeddings.npz, classified.jsonl, raw_clauses.jsonl and serve with python -m http.server; (3) Manual upload below.
Note: Google Drive links don't work directly (no CORS). Re-host on GitHub Releases, jsDelivr, S3 with CORS, or a same-origin folder.
